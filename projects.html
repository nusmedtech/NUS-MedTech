Our Projects

Find out what inspiring and innovative projects are going on at NUS Medtech!

Project DAIMS

DAIMS, Disease-Agnostic Infection Monitoring System, aims to create an infrastructure for real-time monitoring of infectious disease spread.


Project Hormonique

Hormonique aims to revolutionise women's health and FemTech with the development of a transdermal, colour-changing menopausal patch, which features an adjustable estradiol delivery rate tailored through real-time oestrogen level detection.

Project OneTouch

Project OneTouch aims to help evolve the current incentive spirometer.

Project HESYRA

At Hesyra, we are building a safe end-to-end oncology specific AI workflow co-pilot designed to automate administrative and simple clinical tasks using large language models (LLMs) and medical speech recognition technology.
